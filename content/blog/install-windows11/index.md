## 安装 Windows 11

下载之后解压镜像
双击运行 setup.exe 进行安装
如果你的电脑不满足 Windows 11 的要求,你可以使用 PE 安装
如果你下载了一个自带中文语言包的镜像,那么你看到的就是中文的安装界面
按照步骤安装完成后应该会跳出 OOBE 界面
在这个界面选择 China
然后点击 Yes
然后会提示选择键盘布局

如果你下载了包含中文包的镜像那么这里可以选择中文输入法
之后你要配置账户,我用的是 Microsoft 365 E5 的组织账号
之后按照步骤提示配置完成后就可以了
